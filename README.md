# AI-img-gan_GEN
一个AI生成图片的仓库,核心是生成对抗网络（GAN)
# 🎨 AI_img-gan_gen

> 用 GTX 1050 从零训练一个 AI 图片生成器

这是一个完整的、从零开始的 AI 图片生成项目。  
你不需要买高端显卡，不需要租云服务器，用你手头的电脑就能跑。

---

## 📌 它能做什么？

你给它看 3 万张图片，它学会“这类图片长什么样”。  
训练完成后，它能生成一张全新的、不存在的图片。

本项目默认支持三种风格：
- 赛博朋克
- 可爱
- 古风

## 📦 快速开始

```bash
# 1. 克隆项目
git clone https://github.com/你的用户名/AI_img-gan_gen.git
cd AI_img-gan_gen

# 2. 安装依赖
pip install -r requirements.txt

# 3. 爬取图片数据（3万张）
python crawler.py

# 4. 开始训练（128×128，大约需要几周）
python train.py

# 5. 生成一张图片
python inference.py
```

## ⚙️ 参数调优

在 `config.py` 里可以调整：

| 参数 | 作用 | 建议 |
|------|------|------|
| `IMAGE_SIZE` | 图片分辨率 | 64（快）/ 128（清晰） |
| `BATCH_SIZE` | 每批处理数量 | 显存小就改小 |
| `EPOCHS` | 训练轮数 | 100~300 |
| `LEARNING_RATE` | 学习率 | 一般不动 |

---

|--------|------|

## ❓ 常见问题

**Q：生成的图为什么糊？**  
A：默认分辨率是 64×64，可以改到 128×128，但训练时间会变长。

**Q：训练到一半关机了怎么办？**  
A：重新运行 `train.py`，会自动从断点继续。

**Q：显存不够怎么办？**  
A：调小 `BATCH_SIZE`，或调低 `IMAGE_SIZE`。

**Q：爬虫爬不到图怎么办？**  
A：可能是网络问题，按 `Ctrl+C` 暂停，重新运行会继续。
